# Apache Guacamole

Apache Guacamole is a clientless remote desktop gateway that allows you to securely access your systems through a web browser.

Supported protocols include:

- RDP (Windows Remote Desktop)
- VNC
- SSH

No client software is required—simply open Guacamole in your browser and connect to your servers, desktops, or network devices.

## Features

- Browser-based remote access
- Built-in user management
- Multi-factor authentication (TOTP supported)
- Brute-force protection (Auth Ban supported)
- Session recording support
- LDAP, OpenID Connect and SAML authentication available
- Persistent configuration and database storage

## Default Login

Username: `guacadmin`

Password: `guacadmin`

**Important:** Change the default password immediately after your first login.
